# Известные проблемы и решения

## 1. Апострофы в ключах (РЕШЕНО)
**Проблема**: игра использует умные апострофы U+2019 (`'`), ручные переводы — прямые U+0027 (`'`)
**Решение**: `make_final.py` нормализует оба к U+0027 перед сравнением
**Затронуто**: ~1732 строки из ручных write_chunk*.py
**Файл**: `make_final.py` функция `normalize()`

## 2. JSON с незакрытыми кавычками (РЕШЕНО)
**Проблема**: агенты иногда генерировали невалидный JSON когда EN строка содержала кавычки
**Решение**: добавить в промпт агента:
```
ВАЖНО: если строка содержит кавычки " внутри — экранируй как \" в JSON
```
**Затронуто**: progress_050, 056, 057 (пересозданы)

## 3. Строки 100-199 не в базе (РЕШЕНО)
**Проблема**: переводы для строк 100-199 были показаны в чате для оценки, но не сохранены
**Решение**: создан `add_chunk002.py` с этими переводами

## 4. UnityPy не читает type tree из resources.assets
**Проблема**: MonoBehaviours в resources.assets имеют stripped type tree — UnityPy не может читать поля через read_typetree()
**Решение**: использовать get_raw_data() → binary parse → set_raw_data()
**Файл**: `patch_dialogues_v2.py`

## 5. Шрифт (РЕШЕНО — не нужен патч)
**Проблема**: Предполагалось что TMPro шрифты не содержат кириллицу
**Решение**: Шрифт NotoSansKR уже поддерживает кириллицу! Патч шрифтов НЕ нужен

## 6. GameText_Ru.json ключи секций
**Потенциальная проблема**: имена секций в GameText_En.json начинаются с пробела:
```
" GameText_04_EnglishTutorialSection"
```
При патче нужно матчить с учётом пробела. `patch_gametext.py` обрабатывает это.

## 7. bash heredoc не работает в данной среде
**Проблема**: `python - <<'EOF'` не работает
**Решение**: писать скрипты в файлы, потом запускать

## 8. Две папки бандлов (КРИТИЧНАЯ ОШИБКА — РЕШЕНО)
**Проблема**: Патчили бандлы в `aa/Windows/StandaloneWindows64/`, а игра грузит из `aa/StandaloneWindows64/`
**Решение**: Всегда патчить `aa/StandaloneWindows64/`

## 9. CRC в catalog.json ломает игру (РЕШЕНО)
**Проблема**: Модифицированный бандл не проходит CRC проверку → чёрный экран
**Решение**: Обнулить CRC побайтово в ExtraDataString (base64 → UTF-16LE JSON)
**Важно**: Padding пробелами ПЕРЕД `}` — невалидный JSON! Нужно добавлять пробелы ПОСЛЕ `,` и `:`

## 10. Диалоги в ДВУХ местах (РЕШЕНО)
**Проблема**: Патчили resources.assets, но основная база диалогов — `DB_Dialogue_Eternight` в gamemanager бандле (pid=-5833716775174085199)
**Решение**: Патчить "de" поля в обоих файлах (или только в бандле — он приоритетнее)

## 11. Добавление 11-го языка невозможно без патча DLL
**Проблема**: C# код UIPlayerLanguagesSettings хардкодит маппинг 10 индексов → язык. Добавление "Russian" как 11-го не работает — дропдаун показывает, но язык не применяется
**Решение**: Подход "Вариант А" — подмена немецкого (de) на русский. Dropdown: "German"→"Russian", значения "de" полей → русский текст
