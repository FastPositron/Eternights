### Архитектура: Менеджер + 5 агентов-переводчиков

**Основной поток Claude = Менеджер.** Он НЕ переводит сам. Он:
1. Извлекает все строки из бандла
2. Разбивает на чанки по ~150 строк
3. Раздаёт чанки 5 параллельным агентам
4. Контролирует результат каждого чанка (кол-во строк, нет ли пропусков, формат JSON)
5. Если агент вернул ошибку или неполный результат — перезапускает чанк
6. Когда все чанки готовы — собирает в финальные файлы и патчит бандл

**Агенты = переводчики.** Каждый агент:
1. Получает чанк (150 строк EN) + словарь существующих переводов + терминологию
2. Переводит **сам, без внешних API**
3. Сохраняет результат в `output/chunks_ru/chunk_NNN.json`
4. Возвращает статус менеджеру

### Формат чанков

Входной чанк (менеджер создаёт перед запуском агента):
`output/chunks_en/chunk_NNN.json`:
```json
{
  "chunk_id": 42,
  "strings": [
    {"idx": 2100, "en": "You brought this on yourself."},
    {"idx": 2101, "en": "So... who are you?"},
    ...
  ]
}
```

Выходной чанк (агент сохраняет):
`output/chunks_ru/chunk_NNN.json`:
```json
{
  "chunk_id": 42,
  "translations": [
    {"idx": 2100, "en": "You brought this on yourself.", "ru": "Ты сам это заслужил."},
    {"idx": 2101, "en": "So... who are you?", "ru": "Итак... кто ты такой?"},
    ...
  ]
}
```

## Технические детали

### Как извлечь диалоги из бандла
```python
import UnityPy, struct, re
bundle = '.../aa/StandaloneWindows64/gamemanager_...bundle'
env = UnityPy.load(bundle)
# Найти объект DB_Dialogue_Eternight pid=-5833716775174085199
# В raw data искать паттерн b'\x0d\x00\x00\x00Dialogue Text'
# После каждого — read_unity_str() для EN значения
```

### Как патчить "de" поля
Ищем `b'\x02\x00\x00\x00de\x00\x00'` в регионе каждого Dialogue Text.
Заменяем значение (Unity string: [4-byte len][utf8][padding]) на русский текст.
Замены применяем от конца к началу (чтобы не сбить офсеты).

### Правильные пути (ИГРА ГРУЗИТ ОТСЮДА!)
```
Eternights_Data/resources.assets
Eternights_Data/StreamingAssets/aa/StandaloneWindows64/gamemanager_...bundle  ← НЕ Windows/!
Eternights_Data/StreamingAssets/aa/catalog.json
```

### CRC в catalog.json
- Побайтовая замена в base64 → UTF-16LE JSON
- CRC обнуляем (заменяем цифры на 0, количество цифр должно совпадать)
- Размер бандла обновляем (тоже побайтово, кол-во цифр должно совпадать)
- Для выравнивания: пробелы ПОСЛЕ `,` и `:` (не перед `}` — это ломает JSON парсер Unity)
